Webbutveckling 1-2

Exempel och övningar

Det här repot samlar ett antal exempel över vanliga webbkomponenter och visar hur du kan koda dem. Arbeta dig igenom dina intresseområden och gör tillhörande övningar, eller hitta det du behöver och använd det.

Repot är ett work in progress och hittar du något som är fel kan du skapa ett issue. Om du förbättrar eller skapar något nytt till sidan så skapa en pull request.

Den här sidan är ett komplement till kurswebben, Webbutveckling.jensa.xyz.

För att köra de exempel som finns på den här sidan kan du antingen bygga alltihopa med npm, kolla på sidan i din webbläsare (som du gör nu) eller klona repot och öppna exempel med Live server från Visual studio code.

En del av sidorna innehåller instruktioner för hur du går tillväga för att koda och återskapa dem, men oavsett det så förväntas du inspektera och undersöka källkoden på sidorna. Så öppna utvecklarverktygen i webbläsaren (F12) och se till att Inspektera dokumentet.

Alla sidorna innehåller en eller flera övningar för dig att arbeta med. Koda och läs.
